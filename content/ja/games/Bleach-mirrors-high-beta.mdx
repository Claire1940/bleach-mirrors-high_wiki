<Section icon={Swords} title="キャラクター＆斬魄刀カスタマイズシステム">

**Bleach mirrors high beta**の際立った特徴の一つは、オリジナルキャラクター（OC）における前例のないレベルのパーソナライズ機能です。プレイヤーはあらかじめ用意されたヒーローに制限されることはありません。代わりに、独自の死神を作成し、好みのプレイスタイルに合わせて武器を仕立てることができます。カスタマイズ項目は、外見から斬魄刀の細部に至るまで多岐にわたります。

| カスタマイズカテゴリ | 選択可能なオプション | ゲームプレイへの影響 |
| :--- | :--- | :--- |
| **斬魄刀の柄** | 複数の形状と巻き方 | ビジュアルの個性とグリップのスタイル |
| **鞘のデザイン** | 伝統的、装飾的、またはモダン | 待機モーション中のコスメティック演出 |
| **オーラエフェクト** | 色分けされた霊的エネルギー | 属性の適合やパワーレベルを示す |
| **装飾** | チャーム、タッセル、鍔（つば） | 純粋なビジュアル面のパーソナライズ |
| **アバターの特徴** | 髪型、目、顔のタトゥーや傷 | カットシーンにおけるキャラクターの存在感を定義 |

ビジュアルだけでなく、斬魄刀のカスタマイズメニューは、将来的に異なるコンポーネントが特定の「属性アップグレード」に結びつく可能性を示唆しています。ベータ期間中、プレイヤーはさまざまなカラーパレットや、シリーズの高レアリティキャラクターアートでおなじみとなった黒いエネルギーの水彩画スタイルなどのエフェクトレイヤーを試すことができます。

カスタマイズ要素は多岐にわたりますが、一部のハイティアなオーラエフェクトにはベータ専用の通貨が必要です。ベータ終了までにカスタマイズ予算を最大化できるよう、「デイリーソウル訓練」ミッションを必ず完了しておきましょう。

<Section icon={Target} title="戦闘メカニズムとステータス詳細">

**Bleach mirrors high beta**の戦闘は非常にスピーディで、「アーツカード」のデッキに依存しています。各カードは、標準的な物理打撃から強力な霊術（鬼道）攻撃まで、異なるタイプのアクションを表しています。ダメージを軽減しコンボを繋ぐ主な手段がキャラクターの入れ替え（カバーチェンジ）であるため、3人編成のチーム管理が不可欠となります。

**スタイリッシュなリンクアタックの実行方法：**

<Steps>
  <Step num="1" title="ゲージを貯める">
    通常攻撃で敵と交戦し、霊圧ゲージを貯めてアーツカードを解放します。
  </Step>
  <Step num="2" title="アーツカードを繋ぐ">
    打撃（Strike）または射撃（Blast）カードを順に選択してコンボを開始します。少なくとも3枚連続でカードを使用できる十分なエナジーがあることを確認してください。
  </Step>
  <Step num="3" title="スイッチを誘発する">
    コンボがアクティブな間に、サブキャラクターのアイコンをタップします。タイミングが合えば、途切れることなく攻撃を引き継いで参戦してくれます。
  </Step>
  <Step num="4" title="リンクを完成させる">
    画面に表示される「チェイン」ボタンを押し、アクティブな両キャラクターによるシネマティックなフィニッシュ技を繰り出します。
  </Step>
</Steps>

育成においては、ステータスページを理解することも同様に重要です。各キャラクターは、チーム内での役割を決定する明確なプロフィールを持っています。

| ステータスタイプ | 説明 | 戦略的重要度 |
| :--- | :--- | :--- |
| **HP** | 総体力値 | 防御型のタンクや前衛キャラクターにとって極めて重要 |
| **攻撃力 (Attack)** | 純粋な物理ダメージ | 打撃（Strike）主体のキャラクターにおける最重要ステータス |
| **防御力 (Defense)** | 被ダメージの軽減 | 高レベルのベータボスを生き抜くために不可欠 |
| **霊圧 (Spirit Pressure)** | リソースの生成速度 | アーツカードを連打できる頻度を決定する |
| **物理防御 (Physical Def)** | 近接攻撃への耐性 | 更木剣八のようなキャラクターと対峙する際に重要 |
| **霊術防御 (Spiritual Def)** | 鬼道やエネルギー攻撃への耐性 | 遠距離の射撃（Blast）スペシャリストに対抗するために必要 |

<Callout type="info" title="ステータスのスケーリング">
現在のベータ版ビルドでは、霊圧が最も影響力のあるステータスです。生まれつき霊圧の生成率が高いキャラクターは、デッキの回転率が20%速くなり、より頻繁に必殺技（アルティメット）を繰り出すことができます。
</Callout>

</Section>

<Section icon={Sparkles} title="応用メカニズム：ソウルドライブとカウンター">

**Bleach mirrors high beta**を極めたいベテランプレイヤーにとって、ソウルドライブ（Soul Drive）システムは高ランクマッチで勝利を掴むための鍵となります。この状態は、コンシューマー向けアクションゲームに見られる「フォーカスモード」に似ています。発動すると、キャラクターは強化状態に入り、すべてのアーツカードが光り輝き、与えるダメージが大幅に増加します。

<CardGrid cols={2}>
  <Card title="ソウルドライブ状態">
    - **ビジュアルの強化**：キャラクターが激しい霊的エネルギーで光り輝きます。
    - **カードバフ**：この状態中に使用されるすべてのカードは、15%の確率で防御を無視します。
    - **強力な攻撃**：ヒット時に相手の怯み（よろけ）時間が長くなります。
  </Card>
  <Card title="通常状態">
    - **リソース回収**：霊圧ゲージを貯めることに集中します。
    - **防御的な立ち回り**：ダッシュや回避（バニッシュ）を駆使して、大ダメージのコンボを避けます。
    - **戦術的な準備**：キャラクターを入れ替えて、「カバーチェンジ」のタイマーをいつでも使えるようにしておきます。
  </Card>
</CardGrid>

また、ゲームには「カバーチェンジ」と「カウンター」システムも用意されています。相手のアーツカードを予測できれば、乱菊のような防御型のキャラクターを交代で出し、受けるダメージをブロックして仕切り直すことができます。これにより、プレイヤーは必殺技（アルティメット）を放つ前に、相手のキャラクター交代を誘い出すという心理戦が生まれます。

<Callout type="success" title="カウンター戦略">
「カウンター（Counter）」のプロンプトが表示されたら、敵のアニメーションが中間地点に達するのを待ってからカードをタップしてください。これにより成功の猶予が最大化され、フリーコンボのチャンスに繋がることが多くなります。
</Callout>

</Section>
