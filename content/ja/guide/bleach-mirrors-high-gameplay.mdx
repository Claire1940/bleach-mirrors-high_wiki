<Section icon={Swords} title="核心となる戦闘メカニクスとアクションカード">

**Bleach Mirrors Highのゲームプレイ**の核心は、「没入型護廷十三隊アクション」システムにあります。従来のボタン連打型ゲームとは異なり、本作ではプレイヤーがリアルタイムで適切なアクションを選択するという戦略的レイヤーが強調されています。3人のキャラクターで構成されるチームを操作し、キャラクターを切り替えたり、ダイナミックなカードベースのインターフェースを通じて指示を出したりします。

戦闘ループは、アクションカードの生成と活用を中心に展開します。これらのカードは、基本的な斬撃や鬼道から、複雑な防御弾き（パリィ）まで、さまざまな機動を表しています。ハイレベルなプレイで成功を収めるには、敵の動きを予測し、デッキを回して最も効果的な対抗手段を見つけ出す能力が鍵となります。

| メカニクス | 機能 | 戦略的影響 |
| :--- | :--- | :--- |
| **アクションカード** | リアルタイムでキャラクタースキルを決定 | 高：リソース管理とタイミングが必要 |
| **ソウルドライブ** | 連携によるチームベースの強襲 | 極めて高：敵のガードを崩すのに最適 |
| **モータルブロウ** | 豪華な演出を伴う究極奥義 | 最大：一撃で戦闘を終わらせる可能性 |
| **スピードと戦略の融合** | 反射神経と戦術的選択のブレンド | 中：スキルと計画性の両方に報酬 |

<Callout type="tip" title="戦術アドバイス">

<Section icon={Users} title="キャラクター名簿とチームの相乗効果">

**Bleach Mirrors Highのゲームプレイ**において、チームは最大の資産です。本作では、オリジナルの主人公たちと、千年血戦篇で活躍した人気キャラクターたちが魅力的に混ざり合って登場します。各キャラクターは特定の「スタイル」を持っており、これらのスタイルを組み合わせることが、強力なソウルドライブのコンボを解き放つ鍵となります。

主人公の右島至凛（みぎしま しりん）は、プレイヤーがこの世界へ入るための入り口となります。白根左成（しらね さなり）や成Jurinna（なる じゅりんな）といった新しい顔ぶれに加え、黒崎一護や朽木白哉といった伝説的なキャラクターを仲間に加えてチームを補強できます。

| キャラクター | ロールタイプ | シグネチャースタイル |
| :--- | :--- | :--- |
| **右島至凛** | 万能型 | 適応型斬魄刀 |
| **黒崎一護** | 近接DPS | 高速ラッシュ |
| **朽木ルキア** | クラウドコントロール | 氷結による足止め |
| **浦原喜助** | サポート/ユーティリティ | 戦術ガジェット/鬼道 |
| **朽木白哉** | 範囲攻撃 | 多段ヒットの千本桜 |
| **日番谷冬獅郎** | バーストダメージ | 高威力空中コンボ |

<Callout type="warning" title="チーム編成の注意点">
「ダメージ」タイプのみで構成されたチームは、クリア後のコンテンツで苦戦する可能性があります。避けられないダメージを軽減するために、少なくとも1人はユーティリティまたは防御系のアクションカードを持つキャラクターを編成するようにしてください。
</Callout>

キャラクター間の相乗効果は「ソウルリンク」システムで管理されます。共に戦うことでキャラクター間の絆が深まり、ソウルドライブ攻撃のコストが減少したり、チーム全体のバフの持続時間が延びたりします。自分のプレイスタイルに合った最強の編成（メタ）を見つけるために、さまざまな3人組を試すことが強く推奨されます。

</Section>

<Section icon={Sparkles} title="死神アバターのカスタマイズ">

**Bleach Mirrors Highのゲームプレイ**体験における最も革新的な機能の一つは、アバターカスタマイズの深さです。プレイヤーは単に既存のキャラクターを操作するだけでなく、護廷十三隊の一員として自分自身の未来を切り拓いていくことになります。

カスタマイズは「視覚的な美学」と「機能的な戦闘スタイル」に分かれています。斬魄刀は固定された武器ではなく、ストーリー中の選択や成長に費やしたリソースに基づいて進化していきます。

**視覚的 vs 機能的カスタマイズ：**

| 特徴 | 視覚的影響 | 機能的影響 |
| :--- | :--- | :--- |
| **斬魄刀の形状** | 刀身の形や発光が変化 | 攻撃範囲と速度が変化 |
| **戦闘の構え** | 独自の待機/移動アニメーション | 使用可能なアクションカードプールが変化 |
| **死覇装** | 色やアクセサリーのオプション | わずかなステータス耐性を提供 |
| **霊圧のオーラ** | 力の視覚的な激しさ | モータルブロウのダメージ倍率に影響 |

<Steps>
  <Step num="1" title="初期アバター作成">
    顔立ち、髪型、初期の霊圧特性を含むベースの外見を選択します。
  </Step>
  <Step num="2" title="斬魄刀の覚醒">
    刀の主要な属性（炎、氷、雷、または物理）を選択し、初期のスキルツリーを決定します。
  </Step>
  <Step num="3" title="スタイルの専門化">
    護廷十三隊の訓練場でのチャレンジをクリアすることで、高度な戦闘スタイルを解放します。
  </Step>
  <Step num="4" title="ソウルドライブの統合">
    特定のアライ（仲間）とソウルドライブを発動した際にトリガーされる、独自の連携機動を割り当てます。
  </Step>
</Steps>

</Callout>

<Section icon={Map} title="ストーリーの進行と世界の探索">

千年血戦篇の直後を舞台に、物語は空座町で目覚めた一人の魂から始まります。このナラティブは、**Bleach Mirrors Highのゲームプレイ**進行の根幹をなしています。ゲームには、原作の世界観に完璧にフィットしつつ、古参ファンにも新しい発見があるよう監修されたオリジナル脚本が採用されています。

進行に合わせて、瀞霊廷、虚圏（ウェコムンド）、現世といった象徴的な場所を移動します。各ゾーンには特定の「巡回」ミッションがあり、クリアすることで斬魄刀のアップグレード素材やキャラクターの経験値を獲得できます。

**ミッションタイプの概要：**

| ミッションタイプ | 主な目的 | 報酬 |
| :--- | :--- | :--- |
| **ストーリーチャプター** | メインシナリオを進める | 新キャラクター/場所の解放 |
| **隊舎試練** | 隊長級のAIを撃破する | 上位ティアのアクションカード |
| **虚（ホロウ）狩り** | 敵のウェーブを殲滅する | 装備用のクラフト素材 |
| **PvP決闘** | 他のプレイヤーと腕を競う | シーズン限定のコスメティック報酬 |

<Callout type="info" title="設定ノート">
ストーリーでは、大戦後に生じた権力の空白を探索し、新旧の死神たちの協力が必要となる新たな脅威を描きます。今後のキャラクターアークのヒントについては、2026年のアップデートで公開される「キャストコメント」に注目してください。
</Callout>

<Section icon={Zap} title="高度な戦闘戦略">

**Bleach Mirrors Highのゲームプレイ**の基本をマスターしたら、リーダーボードで上位を目指すための高度なテクニックを活用し始めましょう。その一つが「カードキャンセル」です。これは、高速アクションカードを使用して、より遅く重い攻撃のアニメーションを中断し、シームレスなコンボを繋げるテクニックです。

もう一つの重要な戦略は「ソウルドライブ」のタイミングです。敵が「怯み（スタッガー）」状態にある瞬間にソウルドライブを発動すると、ダメージ倍率が50%増加します。これは、2026年のエンドゲームレイドに登場する高HPのボスを倒すために不可欠です。

| 戦略 | 実行方法 | メリット |
| :--- | :--- | :--- |
| **パーフェクトパリィ** | 衝撃の瞬間に防御カードを合わせる | アクションゲージを25%回復 |
| **属性チェイン** | 氷、次に雷のカードを順に使用 | 「超伝導」スタン効果を誘発 |
| **スイッチイン攻撃** | モータルブロウ中にキャラを切り替える | コンボメーターを維持し圧力を高める |

<Callout type="tip" title="習熟への提案">
各キャラクターの「モータルブロウ」のアニメーションを注意深く観察してください。一部のキャラクターはアニメーションが短く、素早く操作を再開できるため、「タイムアタック」チャレンジに適しています。
</Callout>
