<Section icon={Map} title="没入型護廷十三隊アクションジャンルの定義">

**Bleach Mirrors Highのジャンル**は、バンダイナムコによって公式に「没入型護廷十三隊アクション」と表現されています。この分類は、リアルタイムの戦術的判断と高精細な3D表現を強調することで、従来のターン制ガチャタイトルと本作を明確に区別しています。ゲームプレイはモバイルデバイスに最適化された縦画面（ポートレート）で展開され、アニメ『BLEACH』の視覚的な激しさを維持しつつ、スムーズな片手操作を可能にしています。

**ビデオハイライト:**
- **ビジュアルの忠実度**: 一護、ルキア、そして護廷十三隊の高品質な3Dモデルを紹介。
- **戦闘の流れ**: カードベースのスキルシステムとキャラクター交代メカニクスのデモンストレーション。
- **カスタマイズ**: アバタークリエイターと武器改造インターフェースの初公開。
- **PvPアリーナ**: 対戦ランクモードと連勝報酬のプレビュー。

このジャンルの「没入型」という側面は、プレイヤーが自分自身を投影する主人公としての役割を指します。既存のキャラクターを集めるだけの従来のゲームとは異なり、本作ではカスタムした死神が物語の物語の中心に位置づけられます。「護廷十三隊アクション」とは、護廷十三隊の複雑な階層と能力をコアとなるゲームプレイのループに統合した戦闘の規模を意味しています。

<Section icon={Swords} title="戦闘メカニクスとバトルシステム">

**Bleach Mirrors Highのジャンル**の核心は、スピード感とカード主導の戦略を融合させたバトルシステムにあります。プレイヤーは3人のキャラクターでチームを編成し、画面下の「アクションカード」のデッキを通じて移動やスキルの実行を行うリアルタイム戦闘に挑みます。各カードは、標準的な打撃から強力な鬼道まで、特定の技を表しています。

| メカニクス | 説明 | 主なメリット |
| :--- | :--- | :--- |
| **アクションカード** | 画面下部に表示されるスキルカード | 戦略的なスキルの連携を可能にする |
| **モータルブロウ** | 大ダメージを与えるシネマティックな究極技 | 戦況を一変させるフィニッシャー |
| **ソウルドライブ** | 全チームメンバーによる連携攻撃 | 高いバーストダメージとコンボの延長 |
| **カバーチェンジ** | 敵のコンボ中にキャラクターを交代 | ダメージを軽減し、状況をリセットする |
| **カウンターメカニクス** | 正確なタイミングでのガードや回避 | 高いスキルを要する反応プレイへの報酬 |

戦術的な深みは「ターゲットブレイク」システムによってさらに強化されます。敵の特定の弱点に攻撃を集中させることで一時的に気絶させ、チーム全員でのソウルドライブのチャンスを作り出すことができます。

<Callout type="warning" title="エネルギー管理">
アクションカードの使用には、自動回復するエネルギーゲージを消費します。高コストのカードを乱発するとカウンターに対して無防備になるため、低コストの牽制と高コストのフィニッシャーのバランスを取ることが不可欠です。
</Callout>

</Section>

<Section icon={Trophy} title="進行システムと対戦メタ">

**Bleach Mirrors Highのジャンル**における進行は、カスタムアバターの強化と、「ソウル召喚」（ガチャ）システムを通じて強力な原作キャラクターを収集するという二重のパスをたどります。対戦プレイはアリーナを中心に行われ、プレイヤーはリアルタイムのPvPバトルで競い合います。

PvPは非常に競争が激しく、リーグ昇格システムを採用しています。プレイヤーは「一般隊員」からスタートし、バトルポイント（BP）を稼いでランクを上げます。上位ランクでは、限定称号、レアなカスタマイズアイテム、高級通貨などのより豪華な報酬が得られます。

| PvPランク | 必要BP | 月間報酬 |
| :--- | :--- | :--- |
| **隊員 (Officer)** | 0 - 1,000 | 100 霊玉、基本称号 |
| **副隊長 (Lieutenant)** | 1,001 - 5,000 | 500 霊玉、レアな鍔 |
| **隊長 (Captain)** | 5,001 - 15,000 | 1,500 霊玉、限定衣装 |
| **総隊長 (Head Captain)** | 上位100名 | 5,000 霊玉、ユニークなオーラエフェクト |

<Callout type="warning" title="重複システム">
ハイレベルなプレイには、キャラクターの重複（被り）の獲得が不可欠です。重複は基本ステータスを向上させるだけでなく、「Exカラー」のアートワークやモータルブロウの別アニメーションを解放します。
</Callout>
