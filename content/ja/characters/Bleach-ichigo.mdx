<Section icon={User} title="黒崎一護の起源">

黒崎一護は、幽霊が見えるという稀な能力を持つ高校生としてその旅を始めました。彼の人生は、虚（ホロウ）との死闘の最中に死神の力を分け与えてくれた死神、朽木ルキアとの出会いによって永遠に変わることになります。この出来事が、世界で最も著名な「死神代行」としての彼の基盤を築きました。屍魂界（ソウル・ソサエティ）で生まれた通常の死神とは異なり、一護は人間の肉体を維持しており、任務の際には死神代行証を使って魂魄の姿へと変身します。

彼の役割は独特で、護廷十三隊の正式な階級制度の外にありながらも、空座町（からくらちょう）の主要な守護者として機能しています。物語が進むにつれ、一護の血筋は死神、滅却師（クインシー）、そして虚（ホロウ）の力が複雑に融合したものであることが明らかになり、彼を霊界で最も多才かつ予測不能な戦士へと成長させました。

| 属性 | 詳細 |
| :--- | :--- |
| **主な役割** | 死神代行 / 空座町の守護者 |
| **主な所属** | 空座町の仲間たち / 護廷十三隊（名誉） |
| **象徴的な武器** | 斬月（ざんげつ） |
| **核心的な動機** | 家族、友人、および世界の均衡を守ること |

<Callout type="tip" title="役割の洞察">

一護の卍解「天鎖斬月」がユニークなのは、その膨大な霊圧を小さく扱いやすい刀身に凝縮している点にあります。これにより、護廷十三隊のほとんどの隊長格の瞬歩に匹敵、あるいはそれを凌駕する速度での移動が可能になります。千年血戦篇（TYBW）において、一護はこれまでの斬月が自身の真の力の一部に過ぎなかったことを知り、真の二刀流の斬月を鍛え直すことになります。

| 形態名 | 主な外見的特徴 | 戦闘スタイル |
| :--- | :--- | :--- |
| **初期始解** | 柄に晒を巻いた巨大な包丁型の刀 | 重い一撃と遠距離エネルギー放射 |
| **天鎖斬月** | 卍型の鍔を持つ漆黒の細身の刀 | 極限のスピードと連射される月牙 |
| **虚の仮面** | 赤いストライプが入った白い仮面 | 身体能力の大幅な向上と爆発的な力 |
| **完全虚化（大虚形態）** | 角が生え、髪が伸びた異形の姿 | 本能的かつ圧倒的な破壊衝動 |
| **無月（むげつ）** | 黒い霊圧を全身に纏った姿 | すべてを終わらせる一撃必殺の技 |

<Callout type="warning" title="力の制御">

</Callout>

<Section icon={Map} title="ストーリーの年代記と成長の軌跡">

黒崎一護の旅路はいくつかの主要な本編シナリオに分かれており、それぞれが一護の覚悟を試し、霊界の世界観を広げていきます。ルキアとの最初の出会いから、滅却師との最終決戦に至るまで、戦いの規模は常にエスカレートし続けます。

<Steps>
  <Step num="1" title="死神代行篇">
    一護は死神の力を手に入れ、空座町を守り始めます。井上織姫、茶渡泰虎（チャド）、石田雨竜といった重要な仲間たちと出会い、霊的戦闘の基礎を学びます。
  </Step>
  <Step num="2" title="屍魂界（ソウル・ソサエティ）篇">
    処刑のために連行されたルキアを救うため、一護は屍魂界に侵入します。様々な隊長格と死闘を繰り広げて卍解を習得し、最終的に藍染惣右介の反逆を暴く手助けをします。
  </Step>
  <Step num="3" title="破面（アランカル）・虚圏（ウェコムンド）篇">
    藍染率いる破面の軍勢が現世を襲撃。一護はさらわれた織姫を救出するために虚圏へ乗り込み、十刃（エスパーダ）と対峙しながら、虚の仮面の力をマスターしていきます。
  </Step>
  <Step num="4" title="死神代行消失篇">
    藍染との死闘の末に力を失った一護は、「XCUTION（エクスキューション）」と出会います。死神の力を取り戻すために「完現術（フルブリング）」の能力を開発し、裏切り者である銀城空吾と対峙します。
  </Step>
  <Step num="5" title="千年血戦篇">
    滅却師の帝国「見えざる帝国（ヴァンデンライヒ）」の侵攻により、最終決戦が勃発。一護は自身の出生の真実を知り、真の斬魄刀を手に入れて、滅却師の始祖ユーハバッハに立ち向かいます。
  </Step>
</Steps>

<Section icon={Swords} title="象徴的な死闘と戦術分析">

黒崎一護の戦闘史は、アニメ史に残る名勝負の数々で彩られています。それぞれの戦いの中で、彼は自身の技術を進化させ、霊圧の新たな活用法を見出すことを余儀なくされました。朽木白哉との戦いは彼が屍魂界にとって無視できない脅威であることを証明し、ウルキオラ・シファーとの死闘は彼を虚の力の限界（完全虚化）へと追い詰めました。

| 对戦相手 | 戦闘場所 | 使用した主な技 | 結果 |
| :--- | :--- | :--- | :--- |
| **朽木白哉** | 双殛の丘 | 卍解（天鎖斬月） | 引き分け / 白哉の敗北宣言 |
| **グリムジョー・ジャガージャック** | ラス・ノーチェス | 虚の仮面 + 月牙天衝 | 一護の勝利 |
| **ウルキオラ・シファー** | ラス・ノーチェス天蓋上 | 完全虚化（大虚形態） | 一護の勝利（暴走状態） |
| **藍染惣右介** | 空座町（偽） | 最後の月牙天衝（無月） | 藍染の封印 / 一護の死神の力喪失 |
| **銀城空吾** | XCUTIONアジト | 完現術融合・卍解 | 一護の勝利 |
| **ユーハバッハ** | 霊王宮 | 真の卍解 + 静止の銀の矢 | 一護の勝利 |

一護の基本的な戦術は、超高速移動と**月牙天衝**を中心に展開されます。刃の先端に霊圧を凝縮し、それを三日月型の衝撃波として放ちます。卍解状態では、この攻撃の黒いバージョン（黒い月牙 / 黒月牙）を使用することができ、その破壊力は飛躍的に向上します。

<Callout type="info" title="戦闘戦略">
スピードに優れた相手に対しては、一護は自身の反射神経と「瞬歩」を頼りに戦います。更木剣八やノイトラのような防御力の高い敵に対しては、相手の「鋼皮（イエロ）」や霊的な鎧を打ち破るために、月牙天衝の出力を最大化することに集中します。
</Callout>

</Section>

<Section icon={Sparkles} title="能力と戦闘システム">

剣技にとどまらず、一護はその特異な血筋に由来する複数の異なる戦闘システムを駆使します。これらの能力により、彼は異なる霊的種族の技術を併せ持つ「ハイブリッド」な戦士となっています。

- **月牙天衝（げつがてんしょう）**: 凝縮した霊圧を放つ、彼の代表的な遠距離攻撃。
- **瞬歩（しゅんぽ）**: 死神が間合いを一瞬で詰めるために使用する超高速移動術。
- **虚（ホロウ）化**: 仮面を装着することで、身体能力と霊圧を爆発的に引き上げる。
- **完現術（フルブリング）**: 物質に宿る魂を引き出して使役する能力。一護は自身の死神代行証を媒体とします。
- **静血装（ブルート・ヴェーネ）**: 血管に霊子を流し込んで肉体を硬化させる滅却師の防御技術（千年血戦篇で覚醒）。

| 能力タイプ | 技術名 | 主な使用者 | 効果 |
| :--- | :--- | :--- | :--- |
| **死神** | 卍解 | 隊長格の死神 | 斬魄刀の最終解放・形態変化 |
| **虚** | 虚閃（セロ） | 破面 / 一護（完全虚化時） | 凝縮されたエネルギー放射 |
| **滅却師** | 飛廉脚（ひれんきゃく） | 石田雨竜 / ユーハバッハ | 霊子の流れに乗る高速移動術 |
| **完現術** | 物質の魂の使役 | XCUTIONメンバー / 一護 | 物質に宿る魂を引き出し武器化する |

<Callout type="tip" title="極意のアドバイス">
最終決戦において、一護は虚と滅却師の力を融合させ、シリーズ最強クラスの合体技である「王虚の閃光（グラン・レイ・セロ）を纏った月牙天衝」を放つことが可能になります。
</Callout>

</Section>
